Notaciones

Para atacar propiedades en un objeto. 

var futbolista = { 
  nombre: "Cristiano", 
  apellido: "Ronaldo", 
  edad: 30, 
  direccion: {
          pais: "España", 
          ciudad: "Madrid",
          calle: "La Finca"
  },
  club: {
    nombre: "Real Madrid CF",
    fundacion: 1902, 
    cargos: { 
      presidente: {
        presiEjecutivo: "Florentino Pérez", 
        presiDeHonor:   "Alfredo Di Stéfano"
      }, 
      entrenador: "Zinedine Zidane"	  
	}
  }   
};

Notación de punto

Como en Java, accedo usando punto para ir bajando de nivel. 

console.log( futbolista.nombre );    
console.log( futbolista.direccion.pais );    
console.log( futbolista.club.cargos.presidente.presiDeHonor );
futbolista.club.cargos.presidente.presiDeHonor = "Paco Gento";

Pero esto se puede hacer muy largo, como se acaba de ver en la última línea de código.
Así que voy a emplear una variable que apunte a determinada propiedad, y desde ahí ya uso la notación. 

var cargosDelClub = futbolista.club.cargos;
cargosDelClub.entrenador = "Guti";
console.log( futbolista );

Notación de corchete

En la notación de corchete es una cadena la que me indica qué propiedad del objeto estoy empleando.
Esto es muy poderoso, porque el valor se inspecciona durante ejecución. Por tanto, puedo emplear una variable para indicar qué propiedad recuperar. 

console.log( futbolista["nombre"] );    
console.log( futbolista["direccion"]["pais"] );  

var campo = "edad";  
console.log( futbolista [campo] );

Emplear propiedades no definidas en el objeto

Devuelven undefined y eso es problemático a la hora de rastrear un error porque el código no cancela. 

  
var campo = "edad2";  
console.log( futbolista [campo] );      // undefined, edad2 es una propiedad que no existe en futbolista
console.log( futbolista.edad2 );        // undefined, edad2 es una propiedad que no existe en futbolista
