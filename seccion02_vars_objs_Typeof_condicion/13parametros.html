Parámetros de una función

Una función puede recibir como parámetro: un tipo primitivo, un objeto o una función. 

var auxNombre = "Carlos";
function imprimirNombre( pNombre ) {
  console.log( pNombre );
}     
imprimirNombre("Javi");     // Invoco con una variable anónima 
imprimirNombre(auxNombre);  // Invoco con una variable explícita

Las funciones son objetos

Las funciones, realmente, son objetos. Como ellos, se declara su contenido entre {} llaves. se declaran con la palabra reservada function
Una función se invoca por su nombre y con los paréntesis después. OJO, que el detalle de los paréntesis es importante.
Si no se ponen los paréntesis, dado que las funciones son objetos, podría asignar la referencia al objeto/función a una variable. 

function imprimirHola() {
  console.log("Hola");
}     
var salida = imprimirHola();    
var referencia = imprimirHola; 
referencia();                   // hola

Javascript crea un contexto para la función

Lo primero que hace Javascript es buscar la variable que se quiere imprimir en la función dentro del contexto de la propia función.
Si no la encuentra, va a un nivel de contexto padre y así hacia arriba sucesivamente, hasta llegar a un contexto global. 

var a_dentroYFuera_antes = 11;
var c_fuera_antes = 33; 

function imprimirVariables() {
  var a_dentroYFuera_antes = 111;
  var b_dentro_antes = 222;
  console.log(a_dentroYFuera_antes);  // 111
  console.log(b_dentro_antes);        // 222
  console.log(c_fuera_antes);         // 33
  console.log(d_fuera_despues);       // undefined
  console.log(e_enNingunSitio);       // Uncaught Reference ERROR (cancela la ejecución)
}    

var d_fueraDespues = 44;

Procedimientos y funciones en Javascript

En Javascript sólo existen funciones, todos devuelven un valor (aunque aparenten ser un procedimiento).
Lo voy a probar con una función vacía, que devolverá undefined. 

var a = 30;

function funcionVacia() { }
function imprimirVariables() {
  console.log(a);
}    
var salida1 = imprimirVariables();
var salida2 = funcionVacia(); 
console.log(salida1);
console.log(salida2);

Invocar una función antes de su declaración

El primer barrido del intérprete Javascript ya ha reservado memoria para la función, así que saltará al código de la función, lo ejecutará, y no dará error. 

imprimirAdios();
function imprimirAdios() {
  console.log("Adiós");
}
