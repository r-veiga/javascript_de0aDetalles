s2 v10 - Por valor y por referencia

Primitivas

Los valores primitivos se pasan siempre por valor.

Objetos

Los valores primitivos se pasan por referencia. 

var persona1 = { nombre: "María" }; 
var persona2 = persona1;
console.log( persona1.nombre );    // María
console.log( persona2.nombre );    // María

persona1.nombre = "Paco"; 
console.log( persona1.nombre );    // Paco
console.log( persona2.nombre );    // Paco
