s2 v09 - Estª dinámica. Tipos primitivos.

Hay 5 tipos de valores primitivos

Los valores primitivos son datos que no son objetos y que no tienen métodos. Las primitivas son inmutables, no se pueden cambiar.
Hay cinco tipos en Javascript (seis en ECMAScript2015): 

var miNumero    = 10;
var miCadena    = "Roberto";
var miBoolean   = false;
var miUndefined = undefined;
var miNulo      = null;

Escritura dinámica

Las variables de Javascript no están tipificadas. Una variable declarada puede contener cualquier tipo de valor, se puede asignar un valor de un tipo y después asignar a esa misma variable otro tipo diferente. 

miNumero = 15;
console.log(miNumero);                // 15
miNumero = false;
console.log(miNumero);                // false
miNumero = "Cadena de caracteres";
console.log(miNumero);                // "Cadena de caracteres"
miNumero = undefined;
console.log(miNumero);                // undefined

Objetos "wrappers" equivalentes a las primitivas

Objetos String, Number y Boolean (undefined y null no tienen). Su método valueOf() devuelve el valor primitivo.

Objetos

Un objeto viene a ser una colección de datos primitivos o de otros objetos.
Se asigna a una variable definiéndolo con llaves { }
Todo con una "nomenclatura de pares", se escribe un par clave-valor separado por dos puntos. Los diferentes pares se separan por comas. 

var objeto = { num : 12, 
               cad : "Madrid", 
               flg : false, 
               objetoHijo : { hijoNum : 112, hijoCad : "Torrejón", hijoFlg : true }
             }
