La impresión de consola

La impresión de consola se va a usar mucho en este curso. sobre el Global Object. Puede ser que en mi código decida usar un identificador que ya existía previamente, como p.ej Attr que originalmente guarda una función, y entonces será machacado por lo mío. 

<script type="text/javascript">
  alert( miVariable );          // Salta un cuadro de diálogo, detiene la ejecución del programa
  console.log( miVariable );    // 
  console.info( miVariable );   // Mensaje por consola. Icono azul
  console.warn( miVariable );   // Mensaje por consola. Icono amarillo
  console.error( miVariable );  // Mensaje por consola. Icono rojo
<\script>

Manipular propiedades desde consola

Es muy sencillo y también se va a emplear mucho en el curso. Consiste simplemente en que escriba en la consola la asignación que quiera. Puedo ver como la asignación le cambia el valor
