s1 v03 - Mi primer código

Javascript crea el Global Object que me almacena cualquier objeto. Con las herramientas de desarrollador del web browser puedo comprobarlo.

Cómo puedo ver el Global Object

Escribo en la consola Javascript (F12) window y veo ese objeto. Lo reconozco como objeto porque el contenido está entre llaves. Al desplegar el resultado podré ver que almacena muchos valores: son todos los objetos y propiedades Javascript.

Añado un par de propiedades en el Global Object

Simplemente creo dos propiedades en mi código Javascript.

En mi bloque de código escribo: 

<script type="text/javascript">
  var a = 1;
  var b = "Roberto";
<\script>

Como trabajo en un navegador web, el Global Object es window. Recargo la página y podré encontrar dentro a y b
